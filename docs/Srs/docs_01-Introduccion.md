## 1. Introducción  
### 1.1 Propósito 
El propósito de este documento es definir de manera clara y estructurada los requisitos del Sistema de Gestión de Autobuses del ITLA (SGA-ITLA). Sirve como referencia para el análisis, validación y control del sistema.

### 1.2 Público Objetivo 
Dirigido a analistas de sistemas, desarrolladores, personal administrativo del transporte, autoridades institucionales y usuarios finales.

### 1.3 Alcance del Producto 
El SGA-ITLA abarca la gestión operativa y administrativa del transporte institucional. Contempla la validación de usuarios, control de uso, organización de abordaje y gestión de reportes operativos, buscando mejorar el control y eficiencia.

### 1.4 Definiciones y Acrónimos 
* **SGA-ITLA:** Sistema de Gestión de Autobuses del Instituto Tecnológico de las Américas. 
* **ITLA:** Instituto Tecnológico de las Américas. 
* **Usuario:** Persona autorizada (estudiantes y empleados). 
* **Conductor:** Empleado responsable de operar el autobús. 
* **Viaje:** Ejecución de una ruta en un horario definido.
